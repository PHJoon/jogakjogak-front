## 🧷 커밋 컨벤션

| 타입(Type)  | 설명(Description) |
|-------------|-------------------|
| **feat**    | 새로운 기능 추가 |
| **fix**     | 버그 수정 |
| **docs**    | 문서 수정 |
| **style**   | 코드 포맷 변경 (비즈니스 로직에 영향 없음) |
| **refactor**| 코드 리팩토링 |
| **test**    | 테스트 코드 추가/수정 |
| **chore**   | 빌드 업무, 패키지 매니저 설정 등 기타 작업 |
