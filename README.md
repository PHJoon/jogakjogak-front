# 조각조각 JOGAKJOGAK 

> AI가 함께하는 나의 취업 성공 투두 리스트

```
나의 이력서를 등록하고 AI에게 채용공고 분석을 맡겨보세요.
AI가 채용공고를 분석하여 합격하기 위해 해야할 일들을 알려줍니다.
```

## 🧷 커밋 컨벤션

| 타입(Type)   | 설명(Description)                          |
| ------------ | ------------------------------------------ |
| **feat**     | 새로운 기능 추가                           |
| **fix**      | 버그 수정                                  |
| **docs**     | 문서 수정                                  |
| **style**    | 코드 포맷 변경 (비즈니스 로직에 영향 없음) |
| **refactor** | 코드 리팩토링                              |
| **test**     | 테스트 코드 추가/수정                      |
| **chore**    | 빌드 업무, 패키지 매니저 설정 등 기타 작업 |
